# Drinkd

Drinkd is a demo coffee ordering app optimized for smartphones and tablets.

## Technology

Drinkd is built with:
- Angular 7
- Bulma
- Firebase

## Getting Started

- Clone this repo
- `npm install`
- `npm i @angular/cli`
- `ng serve`
